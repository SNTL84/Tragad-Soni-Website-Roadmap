# 🏫 School Section — Feature Specification

**Project:** Tragad Soni Community Website
**Prepared by:** Milan SNTL84 · [desidevloper.com](https://desidevloper.com)
**Phase:** P3 (Development) · **Priority:** High

---

## 🏫 School Listings

| Feature | Description | User Role | Phase | Priority |
|---------|-------------|-----------|-------|----------|
| Add School | List a school run by / recommended by community members | Admin/Member | P3 | High |
| School Details | Name, address, board (CBSE/GSEB/ICSE), type, fees range | Admin/Member | P3 | High |
| School Photo | Upload school building & facility photos | Admin/Member | P3 | Medium |
| Contact & Admission Info | Principal contact, admission helpline, website link | Admin/Member | P3 | High |
| Filter by Board | CBSE / GSEB / ICSE / IB filter | All | P3 | High |
| Filter by City / Area | Schools near a selected location | All | P3 | High |
| Filter by Medium | Gujarati / Hindi / English medium filter | All | P3 | High |

## 📝 School Listing Card — Fields

| # | Field | Type | Required | Notes |
|---|-------|------|----------|-------|
| 1 | School Name | Text | Yes | Display on card |
| 2 | Board | Dropdown (CBSE/GSEB/ICSE/IB) | Yes | For filter |
| 3 | Medium of Instruction | Dropdown (Gujarati/Hindi/English) | Yes | For filter |
| 4 | School Type | Dropdown (Govt/Private/Trust-run) | Yes | |
| 5 | Address | Textarea | Yes | |
| 6 | City / Area | Text | Yes | For location filter |
| 7 | Fees Range | Text/Range | No | E.g. ₹15k–50k/yr |
| 8 | School Photo | Image upload | No | |
| 9 | Principal Contact | Phone | No | |
| 10 | Admission Helpline | Phone | No | |
| 11 | Website | URL | No | |
| 12 | Classes Offered | Tags (1st–12th) | Yes | |
| 13 | Samaj-affiliated | Checkbox | No | Shows badge |
| 14 | Admin Notes | Textarea | No | Admin only |
