# 📚 Education & Study Alignment — Feature Specification

**Project:** Tragad Soni Community Website
**Prepared by:** Milan SNTL84 · [desidevloper.com](https://desidevloper.com)
**Phase:** P3 (Development) · **Priority:** High

---

## 🏆 Scholarships

| Feature | Description | User Role | Phase | Priority |
|---------|-------------|-----------|-------|----------|
| Scholarship Listings | Post available scholarships from samaj trust & external bodies | Admin | P3 | High |
| Scholarship Application | Students apply online — upload marksheets, income proof | Members | P3 | High |
| Eligibility Criteria | Set minimum marks, income limit, class filters | Admin | P3 | High |
| Application Tracking | Students track status: Pending / Approved / Rejected | Members | P3 | High |
| Past Recipients | List of scholarship winners per year (with consent) | Admin | P4 | Medium |
| Sponsor a Scholarship | Members/businesses can donate to fund scholarships | Member/Donor | P4 | Medium |

## 📝 Scholarship Application Form — Key Fields

| # | Field | Type | Required |
|---|-------|------|----------|
| 1 | Student Full Name | Text | Yes |
| 2 | Class / Standard | Dropdown | Yes |
| 3 | School / College Name | Text | Yes |
| 4 | Board (CBSE / GSEB / ICSE) | Dropdown | Yes |
| 5 | Last Year % / Grade | Number | Yes |
| 6 | Family Annual Income | Number (range) | Yes |
| 7 | Guardian Name & Contact | Text + Phone | Yes |
| 8 | Scholarship Applied For | Dropdown | Yes |
| 9 | Marksheet Upload | File upload (PDF/Image) | Yes |
| 10 | Income Proof Upload | File upload (PDF) | Yes |
| 11 | Bank Account Details | Text | Yes |
| 12 | Gotra / Family Code | Text | Yes |
| 13 | Additional Statement | Textarea | No |
